# CONTROL-ESCOLAR

## Descripción
Sistema integral de gestión y control escolar diseñado para automatizar los procesos académicos, administrativos y directivos de la institución educativa. La plataforma web permite centralizar la información de la matrícula, optimizar la asignación de horarios, gestionar flujos financieros de pagos y becas, y ofrecer reportes detallados para la toma de decisiones estratégicas de la secretaría académica.

## Objetivo
Desarrollar e implementar una aplicación web modular utilizando el framework Laravel para administrar de forma eficiente los ciclos operativos de alumnos, profesores, personal administrativo y directivos, garantizando la integridad de los datos y la agilidad en los procesos escolares.

## Pila Tecnológica (Stack)
* **Backend:** PHP (Framework Laravel)
* **Frontend:** Blade Templates / JavaScript (archivos de interfaz)
* **Estilos:** Bootstrap / Tailwind CSS
* **Base de Datos:** Relacional (MySQL / PostgreSQL / Oracle SQL)
* **Control de Versiones:** Git & GitHub

## Número de Sprints del Proyecto
El proyecto se dividirá en **3 Sprints** (Metodología Ágil):
* **Sprint 1:** Configuración del entorno (Laravel), creación de ramas y maquetación de la vista **HOME** junto con el módulo de Alumnos y Profesores.
* **Sprint 2:** Desarrollo de los módulos Administrativos (Nivel A y Nivel B) e integración del control de pagos y materias.
* **Sprint 3:** Módulo de Director de Carrera, reportes del Secretario Académico, pruebas de integración y despliegue.
